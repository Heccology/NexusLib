# Nexuslib version 1.2.0

## Changes
- Added a contributor cape switcher settings option (thanks to Yirmiri)
  - Those who have contributor capes can now enable/disable/switch between available capes in a new menu item in "Skin Customization"
  - Added contributor capes "Subterrous" and "Subterrous Super"
- Added NLFluidInteractionRegistry
  - For defining interactions of a fluid when spreading, such as when cobblestone is made when water and lava meet
  - Supplies the fluid type, level, and block position, should return true (interaction was a success), or false
- Added NLDripstoneConversionRegistry
  - For adding custom dripstone block conversion interactions
  - Takes in the input block, the fluid type (what kind of particles the pointed dripstone should drip) and the resulting block state
